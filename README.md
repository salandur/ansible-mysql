salandur.mysql
============

[![Build Status](https://travis-ci.org/salandur/salandur.mysql.svg?branch=master)](https://travis-ci.org/salandur/salandur.mysql)

Ansible role to manage MySQL

### Example:

```yaml
- hosts: all

  roles:
    - salandur.mysql
```
